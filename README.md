# dithering-server
A server to dither, resize images with uploading functionality to another server


## Quick start

1. Clone the repository:

```bash
git clone https://github.com/viraj-bookanna/dithering-server.git
cd dithering-server
```

2. Install Python dependencies:

```bash
# It's recommended to use a virtual environment, e.g.:
python -m venv .venv
source .venv/bin/activate  # Linux / macOS
# On Windows use: .\.venv\Scripts\Activate.ps1  (PowerShell) or .\.venv\Scripts\activate (cmd)

pip install -r requirements.txt
```

3. Run the server:

```bash
# Linux / macOS
python3 server.py

# Windows (PowerShell / cmd)
python server.py
```

If you need a different Python executable or virtual environment, adjust the commands accordingly.
